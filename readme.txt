Displays wordpress comment count in words. Like instead of 5 Comments, it shows Five Comments.

== Description ==

Wordpress Comment count in words plugin enables you to display comment count in words. If you have 5 comments received, 

this plugin will say Five comments instead of default numeric style 5 Comments.

== Installation ==

1. Upload `comment-count-words.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Put &lt;?php comment_words(); ?&gt; where you want to show comments count in your theme file.
4. Its done.
